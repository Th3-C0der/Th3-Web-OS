Create Your Own App

To create a custom app, you need two files:

  1. A JSON configuration file (app-config.json)
  2. A JavaScript file (app.js)

1. Create app-config.json:

{
    "name": "Your App Name",
    "version": "1.0.0",
    "author": "Your Name",
    "description": "App description",
    "icon": "gamepad",
    "main": "app.js",
    "type": "game",
    "window": {
        "width": 500,
        "height": 500
    }
}

2. Create app.js:

class YourApp {
    constructor(container) {
        this.container = container;
        this.init();
    }

    init() {
        // Your app initialization code
    }
}

// Initialize when window opens
const initApp = () => {
    const windowId = 'your-app-window';
    const contentDiv = document.getElementById(`${windowId}-content`);
    if (contentDiv) {
        const app = new YourApp(contentDiv);
    }
};

if (document.readyState === 'complete') {
    initApp();
} else {
    window.addEventListener('load', initApp);
}

3. Install Your App:

  1. Open App Management
  2. Click "Install New App"
  3. Drag and drop both files
  4. Click "Install"
